HOW TO DEPLOY TO OTHER COMPUTERS
================================================================================

METHOD 1: USB DRIVE (Simplest)
-------------------------------
1. Copy PrintServer-Portable folder to USB drive
2. Plug USB into target computer
3. Copy folder to C:\PrintServer
4. Double-click Start-PrintServer.bat
5. Done!

Time: 2 minutes per computer


METHOD 2: NETWORK SHARE
------------------------
1. Share PrintServer-Portable on network:
   \\server\PrintServer-Portable

2. On each computer:
   - Copy folder from network share
   - Run Start-PrintServer.bat

Time: 3 minutes per computer


METHOD 3: ZIP FILE
------------------
1. Create ZIP:
   Compress PrintServer-Portable folder

2. Distribute via:
   - Email
   - Cloud storage (Google Drive, Dropbox)
   - Download link
   - FTP server

3. Users:
   - Download ZIP
   - Extract
   - Run Start-PrintServer.bat


METHOD 4: CREATE INSTALLER (Optional)
--------------------------------------
For professional deployment, create installer:

1. Download Inno Setup:
   https://jrsoftware.org/isinfo.php

2. Update installer-script.iss with correct paths

3. Compile to create PrintServer-Setup.exe

4. Distribute the installer

Users run PrintServer-Setup.exe:
   - Installation wizard
   - Auto-configuration
   - Start menu shortcuts
   - Firewall setup
   - Uninstaller included

CUSTOMIZATION BEFORE DEPLOYMENT
================================================================================

CHANGE PRINTER NAME:

1. Edit: printServer.js
2. Line 27: const PRINTER_NAME = "YOUR_PRINTER_NAME";
3. Save
4. Run: Build-Standalone.ps1 (rebuild bundle)
5. New bundle will have updated printer name

CHANGE PORT:

1. Edit: printServer.js
2. Line 24: const PORT = 9191;
3. Save and rebuild

REBUILD AFTER CHANGES:

PowerShell:
  .\Build-Standalone.ps1

Or manually:
  1. Update printServer.js
  2. Copy to PrintServer-Portable folder
  3. Redeploy

ENTERPRISE DEPLOYMENT OPTIONS
================================================================================

FOR IT DEPARTMENTS:

Option A: Group Policy Deployment
  - Share PrintServer-Portable on network
  - Create Group Policy startup script
  - Auto-deploys to all computers

Option B: SCCM / Intune
  - Package PrintServer-Portable
  - Deploy via SCCM or Microsoft Intune
  - Monitor installation status

Option C: PowerShell Remote
  - Use PowerShell remoting
  - Copy to target computers
  - Start automatically

Sample PowerShell script:
  $computers = "PC01","PC02","PC03"
  foreach ($pc in $computers) {
    Copy-Item "PrintServer-Portable" "\\$pc\C$\" -Recurse
    Invoke-Command -ComputerName $pc -ScriptBlock {
      Start-Process "C:\PrintServer-Portable\Start-PrintServer.bat"
    }
  }
